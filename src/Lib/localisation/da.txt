Når dette afkrydsningsfelt er markeret, overskrives bestemmelsessted fil ikke, hvis det allerede exists.\r\nInstead en ny fil vil blive oprettet med et unikt nummer tilføjes i slutningen af filnavnet.
Re-monitor recorded videos	Re overvåge optagede videoer
By default, once a source file (Recorded video) has been converted,\r\nthe monitor history will track the filename and not queue the same files again.\r\n\r\nEnable this option to ignore the history and monitor all files and add them to\r\nthe conversion queue, even if they have already been converted.\r\n\r\nThis can be useful sometimes, if the recording software does not generate a\r\nunique filename for each episode/show. It is reccomended to use the\r\nDelete original file option in the Settings page when using this option to avoid\r\nan infinite conversion loop.	Som standard, når en kildefil (optaget video) er blevet konverteret, \r\nthe skærm historie vil spore filnavn og ikke kø de samme filer again.\r\n\r\nEnable denne indstilling for at ignorere historien og overvåge alle filer og tilføje dem to\r\nthe omdannelse køen, selvom de allerede har været converted.\r\n\r\nThis kan være nyttig nogle gange, hvis optagelse software ikke genererer a\r\nunique filnavn for hver episode/Vis. Det er anbefalet at bruge the\r\nDelete oprindelige fil valgmulighed på siden Indstillinger, når du bruger denne indstilling til avoid\r\nan uendelig konvertering loop.
